Welcome to EcoGraph! EcoGraph is an interactive data visualization tool
that showcases environmental and nature-related statistics from
countries around the world. With EcoGraph, users can explore key
indicators such as disasters, temperature, forest area and more---all
through clean, dynamic, and insightful visualizations. Designed to
promote awareness and data-driven action, EcoGraph turns global
environmental data into meaningful insights.





## **Features**

-   **Bar charts** for easy comparison of environmental metrics.

-   **Line charts** to track trends over time.

## **Getting Started**

### **Configure the Database**

1.  **Create a MySQL Database:** Create a MySQL database to store the
    data for the EcoGraph application.

2.  **Update Database Connection Settings:** Open the db.js file located
    in /back-end/ directory of the project. Update the database
    connection settings in this file to match your MySQL database
    configuration.

3.  **Set the MySQL Database:** run tables.sql and insertData.sql file
    located in /Database/DDL script. 

### **Configure ExpressJS and Angular app**

1.  Open the command line prompt in back-end directory and **run npm i
    && npm start**

2.  Open the command line prompt in front-end directory and run **npm i
    && ng serve**
